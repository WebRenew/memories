## Project Summary
A single, high-converting marketing page for "memories.sh", a universal memory layer for coding agents. The project focuses on presenting a premium, dev-native infra tool that provides portable agent configs and universal context storage.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Lucide React (icons)
- Framer Motion (animations)
- Next-Themes (theme management)

## Architecture
- `src/app/page.tsx`: Main marketing page entry point.
- `src/components/`: Modular UI components for page sections.
- `src/app/globals.css`: Design tokens and global styling.
- `src/components/ThemeSwitcher.tsx`: System-default theme toggle.

## User Preferences
- Dev-native, crisp, confident tone.
- Minimal, dev-friendly aesthetic.
- Default to system theme (supports Light/Dark).
- Orchid-violet accents across both themes.
- Modern sans for UI and mono for code snippets.

## Project Guidelines
- No framework-specific mentions in copy.
- Short sentences, strong nouns/verbs, minimal adjectives.
- No stock photos; use diagrams, code blocks, and UI cards.
- Micro-interactions for premium feel.
